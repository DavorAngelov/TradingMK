# TradingMK

**TradingMK** is a web application for buying and selling stocks on the Macedonian Stock Exchange. The project provides a complete solution with **frontend**, **backend**, **SQL database**, and **initialization scripts**.  

---

## Overview

The application allows users to:

- View real-time stock prices
- Check historical stock prices
- Create and manage a personal portfolio
- Submit buy/sell trade requests
- Track the current status of trades

---

## Technologies

- **Frontend:** React.js  
- **Backend:** Spring Boot (Java)  
- **Database:** PostgreSQL  
- **SQL Scripts:** For schema creation and data initialization  
- **Real-time Data:** Python Scraper
